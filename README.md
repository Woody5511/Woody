# 作品集背景

綠藤生醫是台灣保養品牌主打身體沐浴、臉部、頭髮保養等商品，銷售通路由線上官網與九間實體門市組成；
公司將產品分為三大類：核心產品、帶路產品、其他產品；在此案例中主要會透過2021年1-4月與2022年1-4月兩年的數據差異，探討首購顧客購買類別與購買通路的關係，**目的要了解什麼組合新客回購率最高**。


## 文件和資源

總共有5份表單，名稱和用途：
1. Customers : 顧客首次購買的基本資料
2. Orders : 綠藤所有的訂單紀錄
3. OrderDetails : 訂單上的詳細內容
4. Products : 產品名稱與分類
5. Channels : 通路清單
